# 1С Helper — Desktop Bridge для Контур.Маркет

Локальный агент, который ставится на ПК клиента и автоматизирует Контур.Маркет через браузер.
Принимает команды из облака `net1c.ru` и выполняет их как "ручные" действия — добавляет товары,
обновляет цены и остатки в обход ограничений API.

---

## Зачем

API Контур.Маркет даёт только чтение (остатки, документы). Добавлять и редактировать товары через API
нельзя. Этот агент решает задачу — открывает Контур.Маркет в персистентном Chromium-профиле, где
логин пользователя сохраняется, и выполняет действия как обычный пользователь.

---

## Требования

* Windows 10/11, macOS 12+ или Linux с GUI
* Python 3.11+ (или готовый `.exe` из релизов)
* Доступ к интернету
* Аккаунт в Контур.Маркет (в который клиент войдёт вручную один раз)

---

## Установка (из исходников)

```powershell
# 1) Склонируй репозиторий и зайди в папку агента
cd agent

# 2) Создай виртуальное окружение
python -m venv .venv
.\.venv\Scripts\Activate.ps1     # на Windows
# source .venv/bin/activate      # на macOS/Linux

# 3) Установи зависимости
pip install -r requirements.txt

# 4) Установи браузер для Playwright (≈130 МБ)
python -m playwright install chromium
```

---

## Сопряжение с аккаунтом net1c.ru

1. Открой [net1c.ru](https://net1c.ru) → Настройки → вкладка **🤖 Агент**
2. Нажми **"Подключить нового агента"** — появится код из 8 символов (например `K7QM9P4R`)
3. Запусти агента на ПК:
   ```powershell
   python main.py
   ```
   Он запросит код — введи его.

   Или сразу с параметром:
   ```powershell
   python main.py pair K7QM9P4R
   ```

4. При первом запуске откроется Chromium с сайтом Контур.Маркет. **Войди в свой аккаунт вручную**.
   Сессия сохранится в локальной папке — повторно логиниться не нужно.

5. Оставь окно браузера и консоль агента открытыми. Агент каждые 3 сек проверяет задачи из облака.

---

## Команды

| Команда | Что делает |
|---------|-----------|
| `python main.py` | запуск агента |
| `python main.py pair КОД` | сопряжение |
| `python main.py status` | показать текущее состояние |
| `python main.py unpair` | отвязать, удалить токен |
| `python main.py --headless` | запустить браузер в фоне (только после первого логина!) |

---

## Как это работает

1. В облаке `net1c.ru` при сохранении накладной создаются **задачи** (`AgentTask`) для агента.
2. Агент каждые 3 секунды делает `GET /api/v1/agent/tasks/poll` и забирает пачку.
3. Для каждой задачи запускается соответствующий обработчик — открывает нужную страницу Контура,
   заполняет поля, сохраняет.
4. Результат отправляется обратно в облако: `POST /api/v1/agent/tasks/{id}/complete`.
5. На странице "Готово" накладной пользователь видит статус синхронизации.

## Автозапуск (Windows)

Чтобы агент стартовал вместе с Windows:

1. Создай ярлык на `main.py` (правый клик → «Создать ярлык»)
2. Нажми `Win+R` → введи `shell:startup` → Enter
3. Перетащи ярлык в открывшуюся папку

Или используй планировщик задач с триггером «При входе в систему».

## Файлы конфигурации

* **Windows:** `%APPDATA%\net1c-agent\config.json`
* **macOS:** `~/Library/Application Support/net1c-agent/config.json`
* **Linux:** `~/.config/net1c-agent/config.json`

Там хранится `token` (секретный!), `agent_id`, `server_url`.
Если хочешь сбросить — удали папку целиком.

Логи: там же в `logs/agent.log`.

Профиль браузера (с куки Контура): `browser-profile/` в той же папке.

## Безопасность

* Токен агента хешируется на сервере (bcrypt). Только первичный ответ `/register` содержит «сырой» токен.
* Агент общается с сервером только по HTTPS.
* Браузер запускается в изолированном персистентном профиле (не трогает твой основной Chrome).
* Все действия агента логируются в `logs/agent.log` и в панели `net1c.ru → Агент → Задачи`.
* В любой момент можешь **отозвать агента** в веб-интерфейсе — токен станет невалидным.

## Частые проблемы

**«Не залогинены в Контур.Маркет»**  
→ Открой окно Chromium которое запустил агент, войди в Контур.Маркет вручную.
   При следующей задаче агент подхватит сессию.

**«Агент офлайн» в веб-интерфейсе, хотя запущен**  
→ Проверь логи `logs/agent.log`. Возможно заблокирован доступ к `net1c.ru` (файрвол/прокси).

**Задачи висят в «В очереди»**  
→ Агент не запущен или не может дойти до сервера. Проверь что `python main.py` показывает
  «Agent starting...» и нет ошибок.

**Селекторы Контура изменились (форма товара не заполняется)**  
→ Открой `kontur_browser.py` и поправь регулярки в `_fill_product_form`. Логи содержат
  скриншоты на ошибках.

## Разработка

Сборка одиночного `.exe` через PyInstaller:

```powershell
pip install pyinstaller
pyinstaller --onefile --name net1c-agent main.py
```

Результат: `dist/net1c-agent.exe`. Для работы всё равно нужен установленный Chromium
(команда `playwright install chromium`).
